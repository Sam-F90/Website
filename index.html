Recent Projects

Twitter Unfollower

A Python script to automatically unfollow users on twitter, using Twitter's API and the python module Tweepy. The script will unfollow all users currently muted and send a direct message to you after completion of all the accounts unfollowed.

Pokemon Namer

Uses Android Debug Bridge to name pokemon according to their individual values. Uses Pillow to analyze phone screenshots and Google's Tesseract OCR to read text on screenshots.

NBA head to head

A simple program that uses python to scrape basketball reference to compare two player's career stats. it was good practice with erb scraping when an API is not available.
